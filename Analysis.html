Background and Objectives

This markdown file uses ggplot2 (Wickham 2016) and dplyr(Wickham et al. 2020) packages to analyze the Apple Maps mobility data from the time before the COVID-19 pandemic through the initial shutdowns and continuing changes over the course of the year.

The goal is to create a subset to a particular state, tally up the number of cities and counties in that state, and then to make several plots to visualize that data.

The state analyzed in this document is South Carolina.
